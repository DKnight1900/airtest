Airtest
=====
Python lib for **android** and **ios** app test.

在线文档 <http://netease.github.io/airtest>

离线文档使用方法：

	git clone https://github.com/netease/airtest && cd airtest
	gem install jekyll
	git checkout gh-pages
	jekyll serve --baseurl=''
